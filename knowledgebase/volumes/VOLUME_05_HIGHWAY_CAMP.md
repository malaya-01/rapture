# Volume 5 — Highway Camp

## Volume metadata

| Field | Value |
|-------|-------|
| **Number** | 5 |
| **Title** | Highway Camp |
| **Chapters** | 181–250 |
| **Age** | 1 — Survival |
| **Macro arc** | Highway Camp (91–150) / Long Winter begins |
| **Primary theme** | Building |
| **Secondary theme** | Belonging |
| **Purpose** | Community; farms; festivals; first winter; beloved side-character death; **future Sanctuary site discovered** |

**Why this volume matters:** Without it, Sanctuary is a city on a map. With it, Sanctuary becomes **home**. Readers must care about schools, farms, festivals, children before future wars destroy some of them.

**Pacing contract:** No immediate second dungeon. Earn what was gained in Vol 4.
